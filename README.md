# 风电场数据分析与功率预测

基于真实风电场运行数据的机器学习综合实训项目，涵盖数据预处理、可视化分析、聚类工况划分及风电功率预测建模。

## 项目结构

```
SHIXUN/
├── DATE.csv                       # 原始风电场运行数据
├── analysis.py                    # 任务一 & 任务二：数据预处理 + 可视化相关性分析
├── Task3.py                       # 任务三：K-means 聚类分析
├── Task4.py                       # 任务四：风电功率预测（SVR / BP / Linear）
├── wind_analysis.py               # 全任务整合脚本（单文件运行）
├── wind_analysis_v2.py            # 全任务整合脚本（面向对象 + 时间戳输出目录）
├── data_normalized.csv            # 归一化处理后的数据集
├── RW1/                           # 任务一输出：波形图、时序图、散点图
├── RW2/                           # 任务二输出：散点矩阵、风向玫瑰图、相关性热力图
├── RW3/                           # 任务三输出：K值选择图、聚类可视化
├── RW4(SOLO)/                     # 任务四输出：模型对比图、预测效果图
└── task_outputs/                  # 新版输出目录（含时间戳子目录）
```

## 数据说明

| 字段 | 含义 | 单位 |
|------|------|------|
| DATATIME | 采样时间 | — |
| WINDSPEED | 风速 | m/s |
| WINDDIRECTION | 风向 | 度 |
| TEMPERATURE | 温度 | °C |
| HUMIDITY | 湿度 | % |
| PRESSURE | 气压 | hPa |
| WINDPOWER | 有功功率 | kW |

数据采样间隔约 15 分钟，总计约 39,000+ 条记录。

## 任务概览

### 任务一：数据预处理

1. **缺失值处理** — 前向填充法填补短时缺失
2. **物理异常值剔除** — 过滤风速/功率为负值的异常数据
3. **DBSCAN 聚类去噪** — 基于风速-功率二维空间识别并剔除离散噪声点
4. **Min-Max 归一化** — 将所有数值特征归一化到 [0, 1] 区间

输出：`data_normalized.csv`、波形图、时序图、散点图、去噪对比图、归一化散点图

### 任务二：可视化与相关性分析

1. **统计特征表** — 六维特征的均值/标准差/分位数/极差
2. **散点图矩阵** — 六维特征两两可视化（对角线为分布直方图）
3. **风向玫瑰图** — 按 15° 分箱绘制风向频率分布
4. **皮尔逊相关系数热力图** — 定量分析特征间线性相关性
5. **特征筛选** — 以 |r| ≥ 0.2 为阈值，筛选与功率显著相关的特征

### 任务三：K-means 聚类分析

1. **肘部法 + 轮廓系数法** — 对比确定最优聚类数 K
2. **K-means 聚类执行** — 基于筛选后的特征进行工况划分
3. **PCA 降维可视化** — 将高维聚类结果映射到二维主成分空间
4. **风速-功率聚类散点图** — 直观展示不同工况的分布特征

### 任务四：风电功率预测

基于聚类结果，对每一类数据分别训练以下模型：

| 模型 | 说明 |
|------|------|
| SVR | 支持向量回归（RBF 核） |
| BP | BP 神经网络（两层隐藏层，32 神经元/层） |
| Linear | 线性回归 |

评价指标：**MAE**（平均绝对误差）、**RMSE**（均方根误差）、**R²**（决定系数）

## 主要技术点

- **DBSCAN 密度聚类** — 自适应去噪，无需预设聚类数
- **K-means 聚类** — 结合肘部法和轮廓系数法科学选择 K 值
- **PCA 主成分分析** — 高维数据降维可视化
- **多模型对比** — SVR、BP 神经网络、线性回归三种方法对比评估
- **按工况分组建模** — 先聚类划分运行工况，再对每类单独建模预测
